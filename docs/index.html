Introduction

SAPIJS is a simple and fast API consumer for the web, it uses the latest technology and is cross browser compatible.

Getting Started

Download the latest release of SAPIJS form https://github.com/shaktiphartiyal/sapijs

Simply include the file sapi.min.js to your web page like so: 

                                    <script src="sapi.min.js" type="text/javascript"></script>

Usage

GET Request 

                                    sapi.get('https://jsonplaceholder.typicode.com/posts/1').then(response => {
                                        console.log(response);
                                    }, error => {
                                        console.error(error);
                                    });

POST Request 

                                    sapi.post('https://jsonplaceholder.typicode.com/posts/1').then(response => {
                                        console.log(response);
                                    }, error => {
                                        console.error(error);
                                    });
